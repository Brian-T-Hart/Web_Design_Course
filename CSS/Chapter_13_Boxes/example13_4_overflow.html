Overflow

In this example the paragraph width and height are fixed and the text overflows the container. The overflow property is set to scroll. If using a laptop or desktop, you should see scrollbars. If you are using an ipad or phone, you should be able to swipe up to view the content. Other options for overflow are visible, hidden, and auto. The auto value will only display scroll bars when necessary. The hidden value will simply clip the extra content and the user would not see it. With the visible value, the content will be visible outside the container.
