CSS Color

Ways to Apply Color

  • RGB Values
    • The amount of red, blue, and green are expressed as numbers between 0 and 255
    • That means there are 256 × 256 × 256 or 16,777,216 possible color combinations
    • Example: rgb(127, 255, 0) is providing a value of 127 for red, 255 for green, and 0 for blue
  • Hex Codes
    • The hexadecimal system uses the numbers 0 to 9 as well as the letters a to f
    • Hex color values begin with a hashtag, followed by six characters
    • Example: #7FFF00
    • The left two characters represent the red color
    • The middle two characters represent the green color
    • The last two characters represent the blue color
  • Color Names
    • There are a bunch of named colors that you can use to style elements
    • The green colored rgb and hex codes above are Chartreuse

Examples

Name RGB Hex Color
Black rgb(0,0,0) #000000
