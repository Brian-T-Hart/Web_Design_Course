rgba

  • You can set both the color and the opacity using the rgba property
  • The rgba property adds a fourth number for opacity
  • Example: rgba(255, 255, 255, 0.7);
  • It is recommended that you set both rgb and rgba properties so colors will display properly in older browsers that do not recognize rgba.
  • If the browser does not recognize rgba, it will use the rgb values and the result will be the image on the left
