OPACITY

opacity: 1.0;

opacity: 0.5;

Understanding opacity

  • An opaque object does not allow light to travel through it. It is not transparent.
  • If something is transparent, like glass, it allows light to pass through which also allows you to see through it.
  • The opacity property allows you to set the opacity/transparency level of an element.
  • It can take a value from 0.0 to 1.0
  • A value of 1.0 means full opacity and is the default value if one is not provided.
  • A value of 0.5 equates to 50% opacity.
