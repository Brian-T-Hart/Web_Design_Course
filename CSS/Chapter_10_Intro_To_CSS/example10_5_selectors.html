Selectors

Selector Example Meaning
Universal * { } Applies to all elements on the page
Type p { } Matches element names
Class .red { } Applies to all elements with the given class
Id #navigation { } Applies to the element with the given id
Child ul>li { } Matches an element that is a direct child of another
Descendant nav a { } Matches all specified descendants
Adjacent Sibling h1+p { } Matches an element that is the next sibling of another
General Sibling h1~p { } Matches an element that is a sibling of another
